# Водоканал Ростов-на-Дону для Home Assistant

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-41BDF5.svg)](https://github.com/hacs/integration)

Интеграция для Home Assistant, позволяющая получать данные из личного кабинета [Водоканал Ростов-на-Дону](https://lkfl.vodokanalrnd.ru).

## Возможности

- Мониторинг лицевых счетов (баланс, задолженность, начисления)
- Отслеживание показаний счётчиков воды (ХВС / ГВС)
- Просмотр истории платежей
- Отправка показаний счётчиков прямо из Home Assistant
- Поддержка нескольких лицевых счетов

## Установка через HACS

1. Откройте HACS в Home Assistant
2. Нажмите **Интеграции** → меню (⋮) → **Пользовательские репозитории**
3. Добавьте URL: `https://github.com/Mackey-way/hass-vodokanal-rnd`
4. Категория: **Интеграция**
5. Найдите "Водоканал Ростов-на-Дону" и нажмите **Установить**
6. Перезапустите Home Assistant

## Ручная установка

1. Скопируйте папку `custom_components/vodokanal_rnd` в `<config>/custom_components/`
2. Перезапустите Home Assistant

## Настройка

1. Перейдите в **Настройки** → **Устройства и службы** → **Добавить интеграцию**
2. Найдите "Водоканал Ростов-на-Дону"
3. Введите email и пароль от личного кабинета

## Сенсоры

### На уровне лицевого счёта

| Сенсор | Описание |
|--------|----------|
| К оплате | Сумма к оплате (₽) |
| Задолженность | Текущая задолженность (₽) |
| Начислено | Начислено за текущий период (₽) |
| Оплачено | Оплачено за текущий период (₽) |
| Последний платёж | Сумма последнего платежа (₽) |
| Дата последнего платежа | Дата последнего платежа |

### На уровне счётчика воды

| Сенсор | Описание |
|--------|----------|
| Показание | Текущее показание (м³) |
| Потребление | Расход за последний период (м³) |
| Дата показания | Дата последней передачи показаний |

## Сервисы

### `vodokanal_rnd.refresh`

Принудительное обновление данных лицевого счёта.

### `vodokanal_rnd.send_readings`

Отправка показаний счётчика воды.

| Параметр | Описание | Обязательный |
|----------|----------|:------------:|
| `device_id` | ID устройства-счётчика | * |
| `account` | Номер лицевого счёта (альтернатива device_id) | * |
| `value` | Показание в м³ | да |

> Укажите либо `device_id`, либо `account` — одно из двух.

Пример вызова по номеру ЛС:

```yaml
action: vodokanal_rnd.send_readings
data:
  account: "123456789"  # ваш номер ЛС
  value: 137
```

## Автоматизация: автопередача показаний

Если у вас есть счётчик воды, подключённый к Home Assistant (например, [Waterius](https://waterius.ru/)), можно настроить автоматическую передачу показаний по расписанию.

### Пример: передача показаний 23-го числа каждого месяца

```yaml
- id: 'vodokanal_auto_send_readings'
  alias: 'Водоканал: автопередача показаний ХВС'
  description: 'Передача показаний с Waterius на Водоканал 23-го числа'
  trigger:
    - platform: time
      at: '09:00:00'
  condition:
    - condition: template
      value_template: '{{ now().day == 23 }}'
    - condition: not
      conditions:
        - condition: state
          entity_id: sensor.waterius_ch1  # ваш сенсор счётчика
          state: unavailable
  action:
    - action: vodokanal_rnd.send_readings
      data:
        account: '123456789'  # ваш номер ЛС
        value: "{{ states('sensor.waterius_ch1') | float | round(0) | int }}"
    - action: telegram_bot.send_message  # опционально
      data:
        message: >-
          Водоканал: показания ХВС переданы
          ЛС: 123456789
          Показание: {{ states('sensor.waterius_ch1') | float | round(0) | int }} м³
  mode: single
```

### Что нужно изменить в примере

| Параметр | Что подставить |
|----------|---------------|
| `sensor.waterius_ch1` | Entity ID вашего сенсора счётчика воды |
| `123456789` | Номер вашего лицевого счёта Водоканала |
| `now().day == 23` | День месяца для передачи (1-28) |
| `09:00:00` | Время передачи |
| `telegram_bot.send_message` | Ваш сервис уведомлений (или удалите блок) |

### Поддерживаемые счётчики

Автоматизация работает с любым сенсором воды в Home Assistant:

- [Waterius](https://waterius.ru/) (Wi-Fi модуль для импульсных счётчиков)
- Zigbee-датчики расхода воды
- ESPHome с импульсным счётчиком
- Любой `sensor` с `device_class: water` и показаниями в м³
